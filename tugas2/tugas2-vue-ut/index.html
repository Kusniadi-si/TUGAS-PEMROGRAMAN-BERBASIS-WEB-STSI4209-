Selamat Datang di Aplikasi SITTA UT

Sistem integrasi untuk kebutuhan pengelolaan proses pemesanan dan distribusi bahan ajar di lingkungan Universitas Terbuka berbasis Vue 3 Framework.

📚

Stok Bahan Ajar

Kelola daftar inventaris buku, pantau limit status keselamatan stok (safety stock), serta lakukan re-order bahan ajar untuk seluruh UT Daerah secara real-time.

Buka Manajemen Stok
🚚

Tracking Delivery Order

Lacak pengiriman paket bahan ajar mahasiswa, buat input order baru, otomatisasi nomor DO, serta pantau riwayat log perjalanan ekspedisi.

Buka Tracking DO

Panduan Singkat Penggunaan Aplikasi

  • Gunakan menu Stok Bahan Ajar jika ingin memperbarui sisa buku di rak atau menyaring buku yang kritis/habis.
  • Gunakan menu Tracking Delivery Order untuk mendaftarkan resi distribusi pengiriman modul kuliah berdasarkan NIM mahasiswa.
